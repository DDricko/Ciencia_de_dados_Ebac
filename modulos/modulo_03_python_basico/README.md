# Módulo 03 - Python Básico

## Objetivo
Introdução aos conceitos fundamentais da linguagem Python, essenciais para ciência de dados.

## Conteúdo

### Tópicos Abordados
- Variáveis e tipos de dados
- Estruturas de dados (listas, tuplas, dicionários, sets)
- Operações básicas
- Entrada e saída de dados
- Controle de fluxo (if/else)
- Loops (for, while)

### Arquivos

#### Notebooks
- `Profissão_Cientista_de_Dados_M3_Pratique_Python_basico.ipynb`: Material prático com exercícios e exemplos

#### Exercícios
Pasta `exercicios/` contém:
- **Lab01_ex01.py a Lab01_ex05.py**: Exercícios sobre variáveis e tipos de dados
- **Lab02_ex01.py a Lab02_ex03.py**: Exercícios sobre estruturas de dados
- **mod03ex02_instrucao.ipynb**: Exercício instrucional do módulo 03

## Conceitos-Chave

1. **Variáveis**: Containers para armazenar valores
2. **Tipos de dados**: int, float, str, bool
3. **Listas**: Coleções ordenadas e mutáveis
4. **Dicionários**: Pares chave-valor
5. **Tuplas**: Coleções ordenadas e imutáveis

## Como Executar

### Scripts Python (.py)
```bash
python exercicios/Lab01_ex01.py
```

### Notebooks (.ipynb)
```bash
jupyter notebook Profissão_Cientista_de_Dados_M3_Pratique_Python_basico.ipynb
```
