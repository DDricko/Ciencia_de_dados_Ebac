# Módulo 04 - Python Intermediário

## Objetivo
Aprofundar conhecimentos em Python com foco em programação orientada a objetos, funções e bibliotecas essenciais.

## Conteúdo

### Tópicos Abordados
- Funções e programação funcional
- List comprehensions
- Lambda functions
- Programação orientada a objetos
- Módulos e pacotes
- Tratamento de exceções
- Manipulação de arquivos

### Arquivos

#### Notebooks
- `Profissão_Cientista_de_Dados_M4_Tarefa_aula_01_e_02.ipynb`: Tarefas das aulas 1 e 2

- `Profissão_Cientista_de_Dados_M4_Tarefa_aulas_06_a_08.ipynb`: Tarefas das aulas 6 a 8

1. **Funções**: Blocos de código reutilizáveis

4. **Exceções**: Tratamento de erros
5. **Arquivos**: Leitura e escrita de dados
