# Módulo 05 - Estatística

## Objetivo
Compreender os fundamentos estatísticos essenciais para análise de dados e machine learning.

## Conteúdo

### Tópicos Abordados
- Estatística descritiva
- Medidas de tendência central (média, mediana, moda)
- Medidas de dispersão (variância, desvio padrão)
- Distribuições de probabilidade
- Teorema do Limite Central
- Testes de hipótese
- Correlação e covariância

### Arquivos

#### Notebooks
- `Mod5Ex_aulas_01a03.ipynb`: Exercícios das aulas 1 a 3
- `Mod5Ex_aulas_04a05.ipynb`: Exercícios das aulas 4 e 5

## Conceitos-Chave

1. **Média, Mediana, Moda**: Medidas de centro
2. **Variância e Desvio Padrão**: Medidas de dispersão
3. **Distribuição Normal**: Distribuição mais comum
4. **Correlação**: Relação entre variáveis
5. **Probabilidade**: Fundamentos para inferência

## Como Executar

```bash
jupyter notebook Mod5Ex_aulas_01a03.ipynb
```
