# Módulo 07 - Análise de Dados Avançada

## Objetivo
Técnicas avançadas de análise de dados, preparação para machine learning e otimização de pipelines de dados.

## Conteúdo

### Tópicos Abordados
- Feature engineering avançado
- Análise de séries temporais
- Análise multivariada
- Técnicas de encoding
- Normalização e padronização
- Seleção de features
- Validação cruzada
- Preparação de dados para ML

### Arquivos

#### Notebooks
- `Mod07_Tarefa_01.ipynb`: Primeira tarefa - Feature Engineering
- `Mod 07 - Tarefa 02.ipynb`: Segunda tarefa - Técnicas avançadas

## Conceitos-Chave

1. **Feature Engineering**: Criação de variáveis preditivas
2. **Encoding**: Transformação de variáveis categóricas
3. **Scaling**: Normalização de features numéricas
4. **Feature Selection**: Seleção de variáveis relevantes
5. **Pipeline**: Automação do processo de preparação

## Como Executar

```bash
jupyter notebook Mod07_Tarefa_01.ipynb
```

## Técnicas Importantes

### Feature Engineering
- Criação de variáveis derivadas
- Binning/discretização
- Transformações matemáticas
- Agregações temporais

### Encoding
- One-Hot Encoding
- Label Encoding
- Target Encoding
- Frequency Encoding

### Scaling
- StandardScaler (Z-score)
- MinMaxScaler
- RobustScaler

## Bibliotecas Utilizadas
- Pandas: Manipulação de dados
- Scikit-learn: Preprocessing e transformações
- NumPy: Operações numéricas
- Matplotlib/Seaborn: Visualização
