# Módulo 06 - Análise de Dados

## Objetivo
Desenvolver habilidades práticas em análise e manipulação de dados usando Python e suas bibliotecas especializadas.

## Conteúdo

### Tópicos Abordados
- Manipulação de dados com Pandas
- Limpeza e preparação de dados
- Análise exploratória de dados (EDA)
- Visualização de dados
- Agregações e agrupamentos
- Merge e join de datasets
- Tratamento de valores ausentes

### Arquivos

#### Notebooks
- `mod06_tarefa_1.ipynb`: Primeira tarefa - Introdução ao Pandas
- `mod06_Tarefa_2.ipynb`: Segunda tarefa - Manipulação avançada
- `mod06_Tarefa_3.ipynb`: Terceira tarefa - Análise exploratória

## Conceitos-Chave

1. **DataFrames**: Estrutura principal do Pandas
2. **Limpeza de Dados**: Tratamento de missing values e outliers
3. **EDA**: Análise exploratória para insights
4. **Visualização**: Gráficos para comunicar resultados
5. **Aggregation**: Sumarização de dados

## Como Executar

```bash
jupyter notebook mod06_tarefa_1.ipynb
```

## Bibliotecas Principais
- **Pandas**: Manipulação de dados
- **Matplotlib**: Visualização básica
- **Seaborn**: Visualização estatística
- **NumPy**: Operações numéricas

## Fluxo Típico de Análise
1. Carregar dados
2. Explorar estrutura (head, info, describe)
3. Limpar dados (missing values, duplicatas)
4. Transformar dados (criação de features)
5. Visualizar padrões
6. Extrair insights
